# Pallas Crypto

Crate with all the cryptographic material to support Cardano protocol:

- [x] Blake2b 256
- [x] Blake2b 224
- [ ] Ed25519 asymmetric key pair and ECDSA
- [ ] Ed25519 Extended asymmetric key pair
- [ ] Bip32-Ed25519 key derivation
- [ ] BIP39 mnemonics
- [ ] VRF
- [ ] KES
- [ ] SECP256k1
